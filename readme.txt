Ein verspieltes Labyrinth-Captcha für WordPress.

== Description ==

PCD Labyrinth Captcha fügt Formularen ein kleines Labyrinth-Rätsel hinzu.

== Installation ==

1. Upload `pcd-labyrinth-captcha` in den `/wp-content/plugins/` Ordner
2. Plugin im Backend aktivieren
3. Shortcode `[pcd_labyrinth_captcha]` in Formularen verwenden
